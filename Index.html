COME CREARE UN TEMPLATE PER I POST

Di Mario Rossi - Pubblicato il 25-10-2020

La prima cosa da fare è suddividere in blocchi la pagina,identificando le parti comuni da applicare a tutte le pagine

Le varie aree saranno suddivise in

  • Contenuti di intestazione del tag header
  • Contenuti centrali variabili del tag main
  • Contenuti di fine pagina comuni,da inserire nel tag footer

Formattazione del codice

Il testo dell articolo dovrà essere formato per identifcare tutte le parti da selezionare con il CSS.Perciò i passi saranno

  1. Proggettare il markup assegnando ad ogni specifico testo il tag appropiato
  2. Scrivere HTML dai tag esterni a quell interni e dall'alto verso il basso
  3. Applicare gli stili CSS

L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressochè totale
